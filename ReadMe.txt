在线客户群价值评价系统原型 

主要实现如下功能：


一、用户管理模块

1、	用户注册  
2、用户登录  
3、信息修改  
4、注销登录
 
普通用户、管理员；
新用户注册后由管理员审批用户权限；
   

二、指标管理
   
指标查看：查询已有的指标；

指标增加：系统管理员能够对指标进行增加

指标有相应的类型，根据作用来划分为, 当前价值类, 未来价值类。另外，有正向指标和负向指标之分，正向指标和负向指标的标准化处理方式不同

指标修改：修改指标的分类和定义；

指标删除：对已有的指标进行删除；


三、综合评价

1.	选择客户细分方式（1.借款目的、2.所在区域 3.年龄 4.收入水平）——获取数据（这个功能不做，做个界面就行了）

2.	指标数据输入：
输入研究的个案名称，以及个案对应的指标管理中已经定义过的指标数据
理想个案指标定义

3.	关联分析：
标准化处理（表格处理）
指标权重计算（表格输出）
   关联度计算（表格输出）
   关联度排序（折线图和表格输出）

4.	聚类分析
选择聚类的个案，采用Ward系统聚类法进行聚类，输出聚类结果，计算不同类的均值。


四、系统管理（管理员有权限）

1.用户审批（新注册用户审批）

2.权限管理（用户权限管理）

3.使用帮助（随便做一个窗口）
